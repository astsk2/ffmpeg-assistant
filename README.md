## 💡 Что это

**FFmpeg** — лучший в мире инструмент для видео и аудио, но он живёт в командной строке, и обычному человеку им не воспользоваться. **FFmpeg Assistant** даёт всю его мощь через спокойный, понятный интерфейс: вы выбираете **цель** («сжать видео», «вытащить звук», «сделать GIF»), а приложение само собирает правильную команду, проверяет её на безопасность и запускает. Никаких флагов, никакого гугления синтаксиса.

> 🔒 **Всё локально.** FFmpeg встроен в приложение, аккаунт не нужен, ваши файлы никуда не загружаются.

## ⚡ Как это работает — за 3 шага

| 1️⃣ Добавьте файл | 2️⃣ Выберите пресет | 3️⃣ Получите результат |
|---|---|---|
| Перетащите видео/аудио или целую папку | «Сжать», «Вытащить звук», «Запись экрана»… | Готовый файл рядом с исходником или в вашей папке |

Команда FFmpeg собирается у вас на глазах — её видно, можно скопировать, а отдельные параметры (качество, скорость) меняются прямо в строке кликом.

## 🧰 Что умеет

- 🎞️ **Сжатие видео** — H.264 / H.265, баланс «качество ↔ размер» одним ползунком (CRF), быстрые и «маленький файл для отправки» пресеты.
- 🔊 **Звук** — вытащить дорожку в **MP3 / Opus / FLAC (без потерь)**, нормализовать громкость, убрать шум, удалить или заменить звуковую дорожку, свести 5.1 → стерео.
- 🔁 **Конвертация и починка** — переупаковка в нужный контейнер, ремонт «несовместимых» файлов без перекодирования.
- 💬 **Субтитры** — извлечь дорожку субтитров в `.srt`.
- 🖼️ **Картинки и GIF** — HEIC/AVIF → JPG, видео → аккуратный GIF.
- 📂 **Пакетно** — обработать **сразу всю папку** (например, сжать все MP4) одним запуском.
- 🔗 **Склейка** — соединить несколько роликов в один.
- 📱 **Под платформы** — готовые пресеты «для отправки / для веба» с правильными настройками.
- 🛡️ **Безопасно** — каждая команда проходит проверку по белому списку, выход изолируется; ничего деструктивного приложение не выполнит.
- 🕘 **История и метаданные** — список прошлых операций (открыть файл/папку в один клик) и полная информация о выбранном файле (кодек, разрешение, битрейт, длительность).

## 💎 Тариф «Эксперт»

Базовые возможности — **бесплатно**. Тариф **«Эксперт»** 🔑 открывает продвинутое:

- ⚙️ **GPU-кодирование** — аппаратное ускорение **NVIDIA NVENC**, **Intel QuickSync**, **AMD AMF** (приложение само определяет вашу видеокарту и доступные кодеки).
- 🖥️ **Запись экрана** — целого монитора (или выбранного из нескольких) **и отдельного окна** приложения, с захватом/без захвата курсора, с записью системного звука и/или микрофона.
- 👁️ **Режим наблюдения** — указываете папку, и каждый новый файл в ней конвертируется автоматически.
- ✍️ **Полное редактирование команды** — для тех, кто знает FFmpeg: правьте готовую строку как хотите (с той же проверкой безопасности).
- 🧩 **Многошаговые пайплайны** — операции в несколько проходов (например, двухпроходная стабилизация).

> Доступен разовый **14-дневный пробный период**. Ключ выдаётся офлайн (см. ниже).
