## 📥 Установка / Install

Откройте [последний релиз](https://github.com/astsk2/ffmpeg-assistant-releases/releases/latest) и скачайте:

| OS | Файл |
|---|---|
| 🪟 **Windows 10/11 (x64)** | `FFmpeg-Assistant_x.y.z_x64-setup.exe` или `.msi` |
| 🍎 **macOS** | `FFmpeg-Assistant_x.y.z_x64.dmg` (Intel; работает на Apple Silicon через Rosetta) |

> **Windows:** SmartScreen может предупредить о неизвестном издателе — нажмите
> «Подробнее» → «Выполнить в любом случае».
> **macOS:** при первом запуске — правый клик по приложению → «Открыть».
> (Приложение пока не подписано сертификатом разработчика.)

## ✨ Возможности / Features

- 🎯 **Пресеты** — выбираете цель, приложение само строит и запускает FFmpeg-команду.
- 🛡️ **Безопасно** — многоуровневая проверка команд, изоляция выходных файлов.
- ⚡ **Эксперт** — GPU-кодирование (NVENC/QSV/AMF), многошаговые пайплайны, режим наблюдения за папкой.
- 📂 **Режим наблюдения** — каждый новый файл в папке конвертируется автоматически.
- 🔌 **Полностью офлайн** — FFmpeg встроен, ничего не отправляется в сеть.
- 🌍 Русский и English.

## 💎 Бесплатно и тариф «Эксперт» / Free & Expert

Приложение почти полностью **бесплатно**. Тариф **«Эксперт»** открывает продвинутые
пресеты и режим наблюдения. Доступен разовый **14-дневный пробный период**.

загружаются. Проверка лицензии — офлайн.
